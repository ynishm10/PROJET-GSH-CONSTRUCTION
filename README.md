# projet-yanis
j'aimerais faire en sorte que les réponses donné dans le formulaire soit envoyé directement sur la boite mail.

Hey, si tu regardes bien, tu m'as envoyé un email depuis ta boite mail ;)

# Terminé

## 1) Se rendre dans backend
  ```bash
    cd backend
  ```
## 2) initialiser un nouveau projet
  ``` npm
  npm init -y
  ```
## 3) Installer les dépendances
 ``` npm
 npm install express nodemailer cors dotenv
```

## 4)  Lancer le serveur
 ``` npm
  node server.js
```
